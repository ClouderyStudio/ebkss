# 英语学习系统 — 项目总文档

> 合并自：`课堂版系统.md` + `英语基础知识学习系统.md`

---

## 一、项目概览

本系统包含两大子模块：

| 子系统 | 产品定位 | 目标用户 | 核心场景 |
|--------|---------|---------|---------|
| **课堂版 (Classroom Mode)** | 教师课上投屏工具 | 教师 | 快节奏闪卡展示 + 自动朗读 + 知识图谱联动 |
| **基础知识学习系统 (Quiz Mode)** | 课堂5分钟快速检测 | 教师 + 学生 | 六种题型自动出题 + AI评分 + 知识图谱 |

| 层 | 技术 | 用途 |

## 三、课堂版 (Classroom Mode)

### 3.1 三种核心模式

| 模式 | 名称 | 流程 | TTS |
|------|------|------|-----|
| **模式1** | 英语说汉 | 显示英语 → 间隔 → 显示中文 | ❌ |
| **模式2** | 英语说英 | 显示英语 → 朗读 → 间隔 → 显示英文解释 | ✅ |
| **模式3** | 听英语说汉 | 先朗读（无文字）→ 间隔 → 显示中英文 | ✅ |

### 3.2 模式时序示例

**模式2（英语说英）**：
```
0.0s: 显示 "give up"
0.1s: 播放 TTS "give up"
0.8s: TTS 播放结束
1.3s: 显示英文解释 "to quit or stop trying"
3.0s: 自动切换到下一个词
```

**模式3（听英语说汉）**：
```
0.0s: 屏幕显示 🎧 图标（不显示文字）
0.1s: 播放 TTS "give up"
0.8s: TTS 播放结束
1.3s: 显示 "give up / 放弃"
3.0s: 自动切换到下一个词
```

### 3.3 教师控制面板

| 控件 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 开始/暂停 | 按钮 | 暂停 | 控制定时刷新 |
| 模式切换 | 下拉 | 英语说英 | 三种模式 |
| 显示→答案间隔 | 滑动条 | 2秒 | 1-5秒可调 |
| 答案停留时间 | 滑动条 | 1.5秒 | 0.5-3秒可调 |
| 音量 | 滑动条 | 80% | 控制 TTS 播放音量 |
| 语速 | 滑动条 | 0.8 | 0.5-1.5，实时生效 |
| 自动下一题 | 开关 | 开 | 关掉后需手动点"下一个" |
| 全屏 | 按钮 | - | 课堂投屏用 |

> 🔴 **未开发**：课堂版前端展示页面尚未实现。当前只有语料管理功能。

## 四、基础知识学习系统 (Quiz Mode)

### 4.1 六种题型

| 题型 | 代码标识 | 展示方式 | 校验方式 | 是否AI |
|------|---------|---------|---------|--------|
| 固定搭配 | collocation | 句子 + `___` | 规则匹配 | ❌ |
| 中英互译 | translation | 中文 → "英文：___" | 多答案匹配 | ❌ |
| 同义词 | synonym | "x的同义词是？" | 多答案匹配 | ❌ |
| 短语 | phrase | 中文释义 → "英文短语：___" | 规则匹配 | ❌ |
| 词形变化 | morphology | 原词 → "变形：___" | 精确匹配 | ❌ |
| 类似结构 | analogy | 模板 + 关键词 → 造句 | AI评分 | ✅ |

### 4.2 前端检测配置

```javascript
export const QUIZ_CONFIG = {
  autoAdvance: true,           // 自动翻页
  showAnswerDelay: 2000,       // 2秒显示答案（毫秒）
  answerHoldDelay: 1500,       // 答案停留1.5秒
  defaultQuestionCount: 4,     // 每次检测4题
  fullscreen: true,            // 全屏模式
  fontSize: 48,                // 正文字号(px)
  answerFontSize: 36,          // 答案字号(px)
  backgroundColor: '#1a1a2e',  // 背景色
  accentColor: '#ff6b6b'       // 答案高亮色
}
```

## 六、TTS 语音方案

### 6.1 模型配置

| 配置项 | 当前值 | 说明 |
|--------|-------|------|
| 模型 | `qwen3-tts-vd-2026-01-26` | 阿里云 DashScope 声音设计模型 |
| 音色 | `沉稳清晰的女教师声音` | 通过文字描述创建的专属音色 |
| API Key | `DASHSCOPE_API_KEY` | 环境变量配置 |

### 6.2 两步调用流程

**第 1 步 — 创建音色**（仅首次执行，voice_id 缓存到文件）
- 端点：`POST /api/v1/services/audio/tts/customization`
- 模型：`qwen-voice-design`
- 传入 `voice_prompt`（文字描述音色）→ 返回 `voice_id`
- 缓存路径：`public/audio/cache/voice_id_cache.json`

**第 2 步 — 合成语音**
- 端点：`POST /api/v1/services/aigc/multimodal-generation/generation`
- 模型：`qwen3-tts-vd-2026-01-26`
- 携带 `voice_id` → 返回 OSS 音频 URL → 下载并缓存

### 6.3 语速策略

| 场景 | 语速 |
|------|------|
| 默认 | 0.8 |
| 教师手动调节范围 | 0.5 - 1.5 |

### 6.4 音频缓存策略

- 同一文本 + 同一语速 + 同一音色只生成一次
- 缓存路径：`/audio/cache/{text}_{voice_hash}_speed{speed}.mp3`
- `voice_hash` 为音色描述的 SHA1 前 8 位
- 缓存未命中时自动调用 TTS API 生成

## 七、数据库设计

### 7.1 课堂版表

```sql
-- 课堂语料表
CREATE TABLE classroom_corpus (
  id INT PRIMARY KEY AUTO_INCREMENT,
  english VARCHAR(200) NOT NULL,
  chinese VARCHAR(200),
  english_explain TEXT,
  phonetic VARCHAR(100),
  tags JSON,
  group_name VARCHAR(100),
  sort_order INT DEFAULT 0,
  source_key VARCHAR(255),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 课堂知识图谱缓存表
CREATE TABLE classroom_knowledge_graphs (
  id INT PRIMARY KEY AUTO_INCREMENT,
  group_name VARCHAR(100) UNIQUE,
  graph_data JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 7.2 基础知识学习版表

```sql
-- 单元表
CREATE TABLE units (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(100),
  grade_level VARCHAR(20)
);

-- 语法点表
CREATE TABLE grammar_points (
  id INT PRIMARY KEY AUTO_INCREMENT,
  unit_id INT,
  name VARCHAR(100),
  description TEXT,
  notes_content TEXT,
  FOREIGN KEY (unit_id) REFERENCES units(id)
);

-- 语料表（核心）
CREATE TABLE corpus (
  id INT PRIMARY KEY AUTO_INCREMENT,
  unit_id INT,
  grammar_point_id INT,
  question_type ENUM('collocation','translation','synonym','analogy','morphology','phrase'),
  question_text TEXT,
  acceptable_answers JSON,
  template TEXT,
  match_rule VARCHAR(50) DEFAULT 'exact',
  requires_ai BOOLEAN DEFAULT FALSE,
  difficulty TINYINT DEFAULT 1,
  FOREIGN KEY (unit_id) REFERENCES units(id)
);

-- 知识图谱缓存表
CREATE TABLE knowledge_graphs (
  id INT PRIMARY KEY AUTO_INCREMENT,
  grammar_point_id INT,
  graph_data JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (grammar_point_id) REFERENCES grammar_points(id)
);
```

## 八、AI 功能

### 8.1 AI 生成题目

- 端点：`POST /api/ai/generate`
- 模型：`deepseek-ai/DeepSeek-V4-Flash`（硅基流动）
- 提示词覆盖六种题型，要求 JSON 格式输出
- 返回 4-12 道题目，含题型、题目文本、可接受答案、匹配规则

### 8.2 AI 评分（analogy 题型）

- 端点：`POST /api/submit`
- 模型：`deepseek-ai/DeepSeek-V4-Flash`（硅基流动）
- 判断学生造句是否正确
- 返回 `is_correct`、`feedback`、`score`

### 8.3 AI 知识图谱生成

- 课堂版：基于语料分组，AI 生成含 root / inflection / phrase 节点的图谱
- 基础知识版：基于语法点，AI 生成含前置知识、后续拓展、易混淆点的图谱
- 图谱缓存到数据库，首次生成后直接读取

## 十一、测试语料

### 11.1 课堂版首批语料（give 词组）

| 英语 | 中文 | 英文解释 | 音标 |
|------|------|---------|------|
| give | 给 | to provide something | /ɡɪv/ |
| given | 给予（过去分词） | past participle of give | /ˈɡɪvən/ |
| give up | 放弃 | to quit or stop trying | /ɡɪv ʌp/ |
| give out | 分发/耗尽 | to distribute or run out | /ɡɪv aʊt/ |
| give away | 赠送/泄露 | to donate or reveal a secret | /ɡɪv əˈweɪ/ |

### 11.2 基础知识学习版测试数据

**Unit 1 - 现在完成时**

| 题型 | 题目 | 可接受答案 |
|------|------|-----------|
| collocation | I haven't seen him ___ (截止到现在) | yet, up to now |
| translation | 在过去的几年里 | over the past few years, over the last few years |
| phrase | 用before造一个现在完成时的句子 | I have met him before. |
| morphology | strike的过去分词 | struck |

**Unit 2 - strike用法**

| 题型 | 题目 | 可接受答案 |
|------|------|-----------|
| collocation | It ___ sb that + 句子 | struck |
| translation | 我突然想到 | It struck me that, It occurred to me that |
| morphology | strike的过去式 | struck |

**Unit 3 - 连词+分词**

| 题型 | 题目 | 可接受答案 |
|------|------|-----------|
| analogy | 模仿 When asked about..., 用"invite"造句 | When invited to the party... |
| collocation | while ___ me as an introvert | identifying |

**Unit 4 - make相关短语**

| 题型 | 题目 | 可接受答案 |
|------|------|-----------|
| phrase | 辨认出 | make out |
| collocation | 散发出(气味/光) | give off |
