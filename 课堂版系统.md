# 英语学习系统课堂版

## 一、项目概述

### 1.1 产品定位
教师课上投屏工具，快节奏闪卡式展示 + 自动朗读 + 知识图谱联动。不需要学生提交答案，纯展示型课堂辅助工具。

### 1.2 核心价值
- 自动定时刷新展示英语内容
- 定时显示答案（可自定义间隔）
- 支持文本 + 语音朗读（TTS）
- 教师全程可控，专注课堂节奏

---

## 二、技术栈

| 层 | 技术 |
|---|---|
| 前端 | Vue 3 + Vite + Pinia + ECharts |
| 后端 | Node.js (Express) |
| 数据库 | MySQL 8.0 |
| LLM API | 硅基流动 DeepSeek V4+（知识图谱生成） |
| TTS API | 硅基流动 CosyVoice2-0.5B（语音合成） |
| 部署 | 域名 es.cldery.com + CDN 亚太 + 服务器 |

## 三、三种核心模式

| 模式 | 名称 | 流程 | TTS |
|------|------|------|-----|
| **模式1** | 英语说汉 | 显示英语 → 间隔 → 显示中文 | 不朗读 |
| **模式2** | 英语说英 | 显示英语 → 朗读 → 间隔 → 显示英文解释 | ✅ |
| **模式3** | 听英语说汉 | 先朗读（无文字）→ 间隔 → 显示中英文 | ✅ |

### 模式详细时序

**模式2（英语说英）**：
```
0.0s: 显示 "give up"
0.1s: 播放 TTS "give up"
0.8s: TTS 播放结束
1.3s: 显示英文解释 "to quit or stop trying"
3.0s: 自动切换到下一个词
```

**模式3（听英语说汉）**：
```
0.0s: 屏幕显示 🎧 图标（不显示文字）
0.1s: 播放 TTS "give up"
0.8s: TTS 播放结束
1.3s: 显示 "give up / 放弃"
3.0s: 自动切换到下一个词
```

## 四、TTS 语音方案

### 4.1 模型配置
| 配置项 | 值 |
|--------|-----|
| 模型 | FunAudioLLM/CosyVoice2-0.5B |
| 音色 | anna（沉稳女声） |
| API 端点 | https://api.siliconflow.cn/v1/audio/speech |

### 4.2 语速策略
| 场景 | 语速 |
|------|------|
| 首次呈现新词 | 0.7 |
| 复习/快速模式 | 0.9 |
| 教师手动调节范围 | 0.5 - 1.5 |

### 4.3 音频缓存策略
- 同一文本 + 同一语速 + 同一音色 只生成一次
- 缓存路径：`/audio/cache/{text}_{voice}_speed{speed}.mp3`
- 首轮测试前预生成所有音频，避免课堂首次调用延迟

## 六、知识图谱

### 6.1 实现策略
- 首次调用 DeepSeek 生成，之后使用缓存
- 左侧切换单词时，右侧图谱自动高亮当前词相关节点

### 6.2 图谱 JSON 格式示例
```json
{
  "nodes": [
    {"id": "give", "name": "give", "type": "root"},
    {"id": "given", "name": "given", "type": "inflection"},
    {"id": "give_up", "name": "give up", "type": "phrase", "meaning": "放弃"},
    {"id": "give_out", "name": "give out", "type": "phrase", "meaning": "分发/耗尽"},
    {"id": "give_away", "name": "give away", "type": "phrase", "meaning": "赠送/泄露"}
  ],
  "edges": [
    {"source": "give", "target": "given", "label": "词形变化"},
    {"source": "give", "target": "give_up", "label": "词组"},
    {"source": "give", "target": "give_out", "label": "词组"},
    {"source": "give_up", "target": "give_out", "label": "易混淆"}
  ]
}
```

## 七、教师控制面板

| 控件 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 开始/暂停 | 按钮 | 暂停 | 控制定时刷新 |

| 语速 | 滑动条 | 0.7 | 0.5-1.5，实时生效 |
| 自动下一题 | 开关 | 开 | 关掉后需手动点"下一个" |
| 全屏 | 按钮 | - | 课堂投屏用 |

## 八、数据库设计

```sql
-- 语料表
CREATE TABLE corpus (
  id INT PRIMARY KEY AUTO_INCREMENT,
  english VARCHAR(200) NOT NULL,
  chinese VARCHAR(200),
  english_explain TEXT,
  phonetic VARCHAR(100),
  tags JSON,
  group_name VARCHAR(100)
);

-- 知识图谱缓存表
CREATE TABLE knowledge_graphs (
  id INT PRIMARY KEY AUTO_INCREMENT,
  group_name VARCHAR(100) UNIQUE,
  graph_data JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 九、API 接口清单

| 方法 | 路径 | 功能 | 说明 |
|------|------|------|------|
| GET | `/api/corpus?group=give` | 获取语料列表 | 返回英语/中文/英文解释/音标 |
| GET | `/api/tts?text=give up&speed=0.7` | 获取音频 URL | 检查缓存，未命中则调用 TTS API |
| GET | `/api/graph?group=give` | 获取知识图谱 | 首次调用 DeepSeek，后续读缓存 |
